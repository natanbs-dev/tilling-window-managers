## NIRI COMMANDS

### Move a janela para outro monitor
- `Mod+Ctrl+Shift+Left  { move-column-to-monitor-left; }`
- `Mod+Ctrl+Shift+Right { move-column-to-monitor-right; }` 

### move a janela para o lado do mesmo monitor**
- `MOD+ctrl+left`
- `MOD+ctrl+right`

### atalhos de exec
- `mod+R`= menu
- `mod+B`= browser
- `mod+E`=files
- `mod+t`=terminal
- `mod+q`=quit
	
### Foco entre monitores
- `Mod+Shift+Left` → Foca monitor esquerdo
- `Mod+Shift+Right` → Foca monitor direito

## Workspaces
- `Mod+1` a `Mod+5` → Ir para workspace
- `Mod+Ctrl+1` a `Mod+Ctrl+5` → Mover janela para workspace
- `Mod+Page_Up` → Workspace acima
- `Mod+Page_Down` → Workspace abaixo

### Tamanho de janelas
- `Mod+Minus`       → Diminuir largura
- `Mod+Equal`       → Aumentar largura
- `Mod+Shift+Minus` → Diminuir altura
- `Mod+Shift+Equal` → Aumentar altura
- `Mod+F`           → Maximizar coluna
- `Mod+Shift+F`     → Tela cheia
- `Mod+C`           → Centralizar coluna

### Overview e ajuda
- `Mod+O`           → Overview (visão geral de workspaces)
- `Mod+Shift+Slash` → Ajuda com atalhos
